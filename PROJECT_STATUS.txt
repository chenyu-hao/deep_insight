📋 核心需求完成情况
═══════════════════════════════════════════════════════════════════════════════

✅ 从 HN 官方 API 拉取 topstories
   - 支持 3 种榜单: top (最热), best (最佳), new (最新)
   - 官方 API: https://hacker-news.firebaseio.com/v0/

✅ 排名前30/50的新闻收集
   - 灵活的 limit 参数: 1-100 之间任意数值
   - 默认: 30 条 / 最多: 100 条

✅ 热度计算与返回
   - Score (得分) + Descendants (评论数) 组合计算
   - 格式: "99分 · 32条评论"
   - 支持原始数值 (score, descendants) 和格式化热度

✅ API 接口返回给前端
   - 端点: GET /api/hotnews/hn
   - 参数: limit, story_type, force_refresh
   - 返回: JSON 格式，包含完整的新闻数据

✅ 与 TopHub 数据格式统一
   - 共同字段: 10 个 (id, title, url, hot_value, rank, source 等)
   - 特有字段: 互不冲突 (HN: score, descendants; TopHub: platform)
   - 完全兼容，可统一处理

💡 前端集成示例
═══════════════════════════════════════════════════════════════════════════════

JavaScript:
  const data = await fetch('/api/hotnews/hn?limit=30&force_refresh=true')
    .then(r => r.json())
  
  data.items.forEach(item => {
    console.log(`[${item.rank}] ${item.title}`)
    console.log(`热度: ${item.hot_value}`)
  })

Vue 3:
  <script setup>
  import { ref, onMounted } from 'vue'
  
  const items = ref([])
  
  const fetchNews = async () => {
    const data = await fetch('/api/hotnews/hn?limit=30&force_refresh=true')
      .then(r => r.json())
    items.value = data.items
  }
  
  onMounted(fetchNews)
  </script>
  
  <template>
    <div v-for="item in items">
      <h3><a :href="item.url">[{{ item.rank }}] {{ item.title }}</a></h3>
      <p>{{ item.hot_value }} | {{ item.author }}</p>
    </div>
  </template>

React:
  function HNNewsList() {
    const [items, setItems] = useState([])
    
    useEffect(() => {
      fetch('/api/hotnews/hn?limit=30&force_refresh=true')
        .then(r => r.json())
        .then(data => setItems(data.items))
    }, [])
    
    return (
      <div>
        {items.map(item => (
          <div key={item.source_id + item.rank}>
            <h3><a href={item.url}>[{item.rank}] {item.title}</a></h3>
            <p>{item.hot_value} | {item.author}</p>
          </div>
        ))}
      </div>
    )
  }

📞 常见问题快速解答
═══════════════════════════════════════════════════════════════════════════════

Q: API 响应为空怎么办？
A: 添加 force_refresh=true 参数强制刷新

Q: 响应太慢怎么办？
A: 使用缓存（不加 force_refresh），响应会快 25 倍

Q: 支持多少条新闻？
A: 1-100 条，超过会自动限制到 100

Q: 能在生产环境使用吗？
A: 可以，已通过完整的测试验证

Q: 如何显示其他榜单？
A: 改变 story_type 参数 (top/best/new)

Q: 如何定期更新数据？
A: 使用 setInterval 定期调用 API，可以每 5 分钟刷新一次
